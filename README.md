BBC News Topic Classifier (NLP)

This repository contains an automated Machine Learning system that reads news articles and instantly categorizes them into five major topics. The project demonstrates Natural Language Processing (NLP) and Multi-class Classification techniques using Python.

Project Overview
This project utilizes a Multinomial Naive Bayes model to analyze the vocabulary of news stories. By identifying key patterns and word frequencies within the text, the model can accurately distinguish between different types of news content.

The model classifies news into 5 categories:

Business

Entertainment

Politics

Sport

Tech

Tech Stack
Language: Python 3.x

Libraries:

Pandas: Data manipulation and analysis.

Scikit-Learn: Machine Learning and Natural Language Processing.

Algorithm: Multinomial Naive Bayes.

Dataset: BBC News Archive (2,225 articles).

Key Features
Automated Text Cleaning: Includes robust handling for missing values and NaN errors to ensure model stability during execution.

Stop-Word Removal: Automatically filters out common English words (such as "the", "a", "is") to focus the model on high-impact, meaningful keywords.

ML Pipeline: Implements a clean Scikit-Learn Pipeline, combining vectorization and classification into a single, professional code structure.

Instant Prediction: Features a dedicated testing section to verify and classify new, unseen headlines in real-time.

File Structure
BBC_News_Classifier.ipynb: The main Jupyter Notebook containing the end-to-end code.

bbc-news-data.csv: The dataset used for training and evaluating the model.

requirements.txt: A list of necessary Python libraries for environment setup.

Performance
The model achieves high accuracy by focusing on domain-specific keywords—such as "shares" for Business or "coach" for Sport. It is highly optimized for speed and performs exceptionally well even with short text inputs.

How to Use
Clone the repository: Copy the project files to your local machine.

Setup Environment: Ensure you have the required libraries installed via pip install -r requirements.txt.

Open the Notebook: Launch the .ipynb file in Google Colab or a local Jupyter Notebook environment.
